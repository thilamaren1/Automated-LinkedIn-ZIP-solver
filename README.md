# Automated-LinkedIn-ZIP-solver

This is script is built using playwright(automation framework) to solve the daily leetcode ZIP puzzel automatically when the script is run.
Before Running the script in your make sure to include a .env file in your pwd with variables "email" and "pass" to store your linkedin profile's email-ID and passowrd respectively
This script runs entirely locally on your system so no worries about the security of using your email and password for the automation

Prerequiestes:
1. Install all the packages mentioned in the package.json file: # npm install
2. Set up Playwright in your system: # npm init playwright@latest
